# Welcome to IKYEASight Documentation

## About IKYEASight

IKYEASight is a Business Intelligence (BI) and Data Analytics initiative focused on the end-to-end lifecycle of IKYEA businesses' data. This documentation provides guides, setup instructions, and technical references for developing and deploying the IKYEASight platform.

## Quick Start

Get started with IKYEASight by following these guides:

- **[Odoo Installation Guide](OdooInstallation.md)** - Set up Odoo for your IKYEASight deployment
- **[Nginx Reverse Proxy Setup](dev_nginx_dual_server_reverse_proxy.md)** - Configure a dual-server reverse proxy with Nginx

## Key Components

### Odoo ERP Integration

Odoo serves as the core enterprise resource planning system for IKYEA businesses. Our installation guide walks you through the complete setup process.

### Nginx Reverse Proxy

For production environments, we use Nginx as a reverse proxy to handle traffic distribution across dual servers, ensuring high availability and load balancing.
